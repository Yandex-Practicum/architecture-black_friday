# pymongo-api

## Как запустить

Запускаем mongodb и приложение

```shell
docker compose up -d
```

После старта всех контейнеров, запустится доолнительный init-контейнер, который инициализирует кластер MongoDB.

[Скрипт инициализации](./scripts/mongo-init.sh) поочередно подключится к configSrv, shard1, shard2 и mongos-router, выполнит скрипты инициализации кластера и завершит свою работу.

## Как протестировать

Для наполнения кластера тестовыми данныи надо запустить скрипт

```shell
./scripts/mongo-init.sh
```
