# Задание 3. Кеширование

## Как запустить

Запускаем mongodb и приложение

```shell
docker compose up -d
```

Инициализируем конфиг-сервис, шарды, роутер и заливаем данные

```shell
./mongo-init.sh
```

## Как проверить

### Если вы запускаете проект на локальной машине

Откройте в браузере http://localhost:8181
