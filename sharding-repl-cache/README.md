# pymongo-api

## Как запустить

Запускаем mongodb и приложение

```shell
docker compose up -d
```

После старта всех контейнеров, запустится доолнительный init-контейнер, который инициализирует кластер MongoDB.

[Скрипт инициализации](./scripts/mongo-init.sh) поочередно подключится к configSrv, shard1, shard2 и mongos-router, выполнит скрипты инициализации кластера и завершит свою работу.

## Как наполнить базу

Для наполнения кластера тестовыми данныи надо запустить скрипт из директории sharding-repl-cache

```shell
./scripts/mongo-init.sh
```

## Как проверить работоспособность

```bash
curl localhost:8080
```

## Как проверить кэширование

Дважды выполнить запрос к приложению, чтобы убедиться, что кэширование работает. Первый запрос должен занять больше времени, чем второй.

```bash
curl -o /dev/null -s -w "Total time: %{time_total}s\n" localhost:8080
```
