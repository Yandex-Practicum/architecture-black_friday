# pymongo-api

## Как запустить

Запускаем mongodb и приложение

```shell
docker compose up -d
```

После старта всех контейнеров, запустится доолнительный init-контейнер, который инициализирует кластер MongoDB.

[Скрипт инициализации](./scripts/mongo-config.sh) поочередно подключится к configSrv, shard1, shard2 и mongos-router, выполнит скрипты инициализации кластера и завершит свою работу.

Запустить скрипт для заполнения базы тестовыми данными

```shell
./scripts/mongo-init.sh
```

## Как проверить

Откройте в браузере http://localhost:8080. Проверьте распределение данных по шардам, а так же количество реплик.

Перейдите по адресу http://localhost:8080/helloDoc/users. Второй и последующие вызовы должны выполняться быстро (<100мс).
