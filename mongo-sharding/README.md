# pymongo-api

## Как запустить

Запускаем mongodb и приложение

```shell
docker compose up -d
```

Скрипт инициации сервера конфигураций, шард, реплик и роутера. Наполнение базы данными. Вы должны находится в директории mongo-sharding
```shell
./scripts/mongo-init.sh
```

Для проверки распределения записей по шардам и отображению реплик в сетах выполнить скрипт (Вы должны находится в директории mongo-sharding):
```shell
./scripts/mongo-check.sh
```
